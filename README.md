This project aims to predict **customer churn in the banking sector** using machine learning models.  
Customer churn refers to customers closing their bank accounts or stopping services.  
Accurately identifying high-risk customers enables banks to take proactive retention actions and  
optimize marketing resources.

This repository supports the MSc dissertation at emlyon business school, following a  
reproducible and fully documented workflow including:
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model training (LR, RF, XGBoost, CatBoost)
- Class imbalance handling
- Probability calibration
- Cost-sensitive evaluation  
- Profit-driven threshold optimization  
- Model interpretability (SHAP)
